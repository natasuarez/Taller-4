# 🚗 Luxury Cars

Proyecto web desarrollado con **HTML5, CSS3, JavaScript y Bootstrap 5**, enfocado en la venta y exhibición de vehículos de lujo. El sitio cuenta con diferentes módulos para clientes y administradores, además de un sistema básico de autenticación.

---

# 📖 Descripción

Luxury Cars es una aplicación web que permite a los usuarios explorar un catálogo de automóviles de lujo y acceder a diferentes funcionalidades mediante páginas independientes como inicio de sesión, registro, recuperación de contraseña y panel de administración.

Este proyecto fue desarrollado como parte de un taller de Desarrollo Web.
